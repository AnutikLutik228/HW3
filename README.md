Проект про высокие горы мира.

Генератор создает CSV-файл с информацией о горах:

* название
* страна
* высота
* сложность

Аналитик читает CSV-файл и создает HTML-отчет со статистикой.

## Как работает программа

Сначала генератор создает файл:

data/data.csv

После этого аналитик читает CSV-файл и создает:

data/report.html

В HTML-отчете показывается:

* количество строк
* статистика по высоте
* статистика по странам
* статистика по сложности

## Структура проекта

* generator/ — генератор данных
* reporter/ — аналитик данных
* data/ — сгенерированные файлы
* run.sh — bash-скрипт для запуска

## Команды

Собрать генератор:

```bash
./run.sh build_generator
```

Запустить генератор:

```bash
./run.sh run_generator
```

Собрать аналитик:

```bash
./run.sh build_reporter
```

Запустить аналитик:

```bash
./run.sh run_reporter
```

Очистить данные:

```bash
./run.sh clear_data
```

## Результат

После запуска в папке data появляются файлы:

* data.csv
* report.html
